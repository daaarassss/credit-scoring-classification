Прогноз дефолта клиентов банка (Credit Scoring Project)

Описание проекта

Данный проект посвящен разработке модели машинного обучения для предсказания риска невыполнения кредитных обязательств клиентами. На основе исторических данных о заемщиках (доход, кредитный рейтинг, стаж работы и т.д.) модель классифицирует клиентов на две группы: надежные (0) и потенциальные дефолтники (1).
Ключевая цель: Достичь метрики F1-score > 0.5 на тестовой выборке.
Технологический стек
	•	Язык: Python
	•	Библиотеки: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
	•	Алгоритмы: Кастомная логистическая регрессия (Gradient Descent), Sklearn Logistic Regression, Linear Regression.
 
Основные этапы реализации

1. Разведывательный анализ данных (EDA)
	•	Дисбаланс классов: Выявлено, что дефолтные клиенты составляют ~28% выборки.
	•	Очистка данных: Значения > 850 приведены к корректному масштабу. Обнаружена "золотая фича" в Current Loan Amount (значение 99,999,999), указывающая на отсутствие задолженности.
2. Feature Engineering (Генерация признаков)
	•	DataTransformer: Реализован класс для автоматизации предобработки (заполнение пропусков медианами, One-Hot Encoding).
	•	Создание признаков:
	◦	DTI (Debt-to-Income) — отношение ежемесячного долга к доходу.
	◦	ID_High_Loan_Flag — бинарный флаг для обработки технических заглушек.
	◦	Negative_Records — агрегированный показатель финансовых проблем заемщика.
3. Моделирование
В проекте реализован и протестирован кастомный класс MyLogisticRegression, использующий:
	•	Функцию активации Сигмоида.
	•	Метод Градиентного спуска для оптимизации весов.
	•	Threshold Tuning (смещение порога до 0.35) для борьбы с дисбалансом классов.

Выводы
Использование кастомной логистической регрессии с ручной настройкой порога классификации позволило превзойти базовые библиотечные решения. Основной прирост качества дали этапы очистки кредитного рейтинга и выделение сигнальных признаков из аномалий в суммах кредита.
